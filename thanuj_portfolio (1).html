Soil Classification using Deep Learning

Built a complete deep learning pipeline to classify 7 types of soil from images using a CNN trained on GAN-augmented data. Tackled real-world challenges like class imbalance (as few as 52 images per class) using CycleGAN to generate 8,413 synthetic training images. Designed a leakage-free Train/Val/Test split strategy ensuring validation and testing used only real original images.
